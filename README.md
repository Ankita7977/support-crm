# Support CRM System

A simple CRM Ticket Management System built using:

- FastAPI
- SQLite
- SQLAlchemy
- React (Vite)
- Bootstrap

## Features

- Create Ticket
- View All Tickets
- Search Tickets
- Filter by Status
- View Ticket Details
- Update Status
- Add Notes

## Backend

```bash
cd backend
uvicorn main:app --reload
```

## Frontend

```bash
cd frontend
npm install
npm run dev
```
