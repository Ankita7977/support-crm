# Support CRM System

Customer Support Ticketing CRM System

## Tech Stack

Backend:
- FastAPI
- SQLite
- SQLAlchemy

Frontend:
- React
- Bootstrap
- Axios

## Features

- Create Tickets
- Auto Ticket ID
- Search Tickets
- Filter by Status
- Update Ticket Status
- Add Notes
- View Ticket Details

## Run Backend

uvicorn main:app --reload

## Run Frontend

npm install
npm run dev
